# Mapa Interactivo

<span className={styles.aliases}>Mapa geográfico interactivo · Visualización geoespacial · Web map</span>

## Descripción

Presenta información del sistema sobre una representación geográfica dinámica, permitiendo al usuario explorar la distribución o ubicación espacial de entidades con datos de localización asociados. A diferencia de un mapa estático, el mapa interactivo permite al usuario explorar activamente la información mediante operaciones como hacer zoom, desplazarse o seleccionar elementos para ver más detalle.

## Capacidades adicionales

<div className={styles.capBlockOptional}>

<div className={styles.capHeading}>
  <svg viewBox="0 0 16 16" width="16" height="16" fill="currentColor"><path d="M8 1a7 7 0 1 1 0 14A7 7 0 0 1 8 1zm.5 3.5a.5.5 0 0 0-1 0V7.5H4.5a.5.5 0 0 0 0 1H7.5v3a.5.5 0 0 0 1 0V8.5h3a.5.5 0 0 0 0-1H8.5V4.5z"/></svg>
  <span>Capacidades adicionales</span>
</div>

- **Filtrado de entidades:** permite al usuario restringir las entidades visualizadas mediante criterios definidos (categoría, rango de valores, fecha)
- **Resaltado y selección:** destaca visualmente entidades específicas según condiciones, búsqueda o interacción del usuario
- **Búsqueda integrada:** permite buscar una entidad y centrar el mapa en su ubicación
- **Detalle al click:** despliega información adicional de una entidad al seleccionarla en el mapa
- **Capas configurables:** permite al usuario activar o desactivar distintas representaciones (tráfico, satelital, división política)

</div>

## Delimitaciones

<Admonition type="danger" title="Qué no es">

- **No** es una [Consulta](/docs/componentes/consultas): la Consulta tiene como propósito recuperar registros específicos según criterios de búsqueda; el Mapa Interactivo tiene como propósito visualizar la distribución o ubicación geográfica de entidades
- **No** es un [Dashboard](/docs/componentes/dashboard): su objetivo no es presentar métricas ni indicadores agregados sobre el estado de un proceso o negocio

</Admonition>

## Modalidades

- **Mapa de marcadores:** representa entidades individuales como puntos o íconos posicionados según su ubicación. Permite explorar la distribución geográfica de elementos concretos.
- **Mapa de calor (heatmap):** visualiza la densidad o concentración de entidades en zonas geográficas mediante gradientes de color, sin distinguir elementos individuales.
- **Mapa coroplético:** colorea regiones geográficas delimitadas (países, estados, comunas) según el valor de un atributo agregado de las entidades asociadas a cada región.
