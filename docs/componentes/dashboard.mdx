# Dashboard

<span className={styles.aliases}>Panel de control · Panel · Cuadro de mando</span>

## Descripción

Un dashboard es una interfaz de usuario que presenta información consolidada y relevante mediante visualizaciones gráficas y resúmenes, diseñada para facilitar la comprensión rápida del estado de procesos, métricas o indicadores clave. Su función principal es permitir que usuarios autorizados monitoreen y tomen decisiones informadas, minimizando el tiempo necesario para entender qué está ocurriendo. La efectividad de un dashboard depende críticamente de seleccionar los datos correctos a mostrar y elegir las técnicas de visualización apropiadas para comunicar cada tipo de información.

- Presenta información resumida y consolidada proveniente de múltiples fuentes en una vista única
- Utiliza representaciones visuales variadas (gráficos de líneas, barras, indicadores, tablas) para comunicar la información rápidamente
- Se actualiza de forma regular, manteniendo la información vigente para decisiones actuales
- Está diseñado para una audiencia y objetivo específico, mostrando solo la información relevante para ese contexto y rol
- Requiere procesos de cálculo o agregación que transformen los datos brutos en las métricas e indicadores presentados

## Capacidades adicionales

<div className={styles.capBlockOptional}>

<div className={styles.capHeading}>

- **Interactividad:** permite al usuario filtrar, agrupar, profundizar (drill-down) y explorar los datos presentados según sus necesidades inmediatas, sin generar un nuevo documento
- **Exportación de vistas:** permite descargar la vista actual o secciones del dashboard en archivos (PDF, imagen o CSV de los datos subyacentes)
- **Alertas visuales por umbral:** destacan visualmente métricas que superan límites definidos (colores, íconos de advertencia), capturando la atención del usuario sobre situaciones que requieren acción

</div>

## Delimitaciones

<Admonition type="danger" title="Qué no es">

- **No** es un reporte estático exportable, sino una vista dinámica de actualización continua
- A diferencia de un [Reporte](/docs/componentes/reporte), el Dashboard permite reconfigurar la vista sin generar un nuevo documento
- **No** es una [Consulta](/docs/componentes/consultas): una consulta responde una pregunta puntual retornando datos específicos; el dashboard presenta una vista holística de múltiples indicadores simultáneamente

</Admonition>

## Modalidades

- **Dashboard Estratégico:** orientado a la alta dirección, muestra KPIs estratégicos y salud general del negocio con actualización diaria o semanal
- **Dashboard Operacional:** presenta métricas operativas y procesos en curso con actualización en tiempo real o frecuente
- **Dashboard Analítico:** para analistas y especialistas, incluye visualizaciones complejas, tendencias y comparaciones detalladas
- **Dashboard Táctico:** enfocado en objetivos de mediano plazo y departamentales, combina elementos operacionales y estratégicos con actualización regular
