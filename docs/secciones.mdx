# Componentes por Sección del Tablero

La matriz muestra qué componentes del catálogo se vinculan con cada una de las seis secciones del Tablero Digital, y con qué carácter: mandatorio, opcional o sin vínculo.

Un vínculo **mandatorio** significa que el componente siempre toca esa sección por sus capacidades esenciales. Un vínculo **opcional** indica que la toca solo cuando incorpora ciertas capacidades adicionales. Una celda vacía significa que el componente no se vincula con esa sección ni en su forma mandatoria ni en sus variantes opcionales documentadas.

## Cómo leer la matriz

Responde a preguntas como:

- *¿Qué componentes necesito si mi sistema tiene muchas entradas manuales?* → Revisa la columna **Entradas M.**
- *¿Qué componentes producen salidas automáticas?* → Revisa la columna **Salidas A.**
- *¿Qué secciones del tablero toca un Dashboard?* → Recorre la fila de **Dashboard**.
